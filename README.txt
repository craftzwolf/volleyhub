# 🏐 VolleyHub

**VolleyHub** é um aplicativo mobile feito com **React Native + TypeScript (Expo)**, criado para ajudar na organização de equipes de vôlei — gerencie jogadores, monte times, agende jogos e facilite a comunicação entre os participantes.

---

## 🚀 Tecnologias usadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [TypeScript](https://www.typescriptlang.org/)

---

## 📱 Pré-requisitos

- Node.js e Yarn instalados
- Expo CLI:
  ```bash
  npm install -g expo-cli
